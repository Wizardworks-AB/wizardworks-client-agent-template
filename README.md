# ⚠️ Moved — this repo is retired

Agent-template authoring moved into the Fae source on 2026-07-04
(Epic #1978): **[fae-gateway → `agent-templates/`](https://github.com/Wizardworks-AB/fae-gateway/tree/main/agent-templates)**.

- **Getting a template:** Fae portal → your organization → **Agent
  templates** → pick variant (Claude Code / Codex / Generic) and optionally a
  project → download. The Fae connection is generated into the bundle.
- **Editing templates:** edit `agent-templates/` in `fae-gateway` — normal
  PR/review flow; the `fae-platform-publish-agent-templates` pipeline
  publishes to the catalog on merge.

No content here is maintained anymore. History is preserved in this repo;
current content lives in `fae-gateway`.
